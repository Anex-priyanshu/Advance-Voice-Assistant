# 🎙️ Advance Voice Assistant

**AI-Powered Desktop Voice Assistant using Python**

An intelligent voice-controlled desktop assistant developed in Python that automates daily computer tasks, controls Windows applications, performs web searches, retrieves online information, and executes system operations using natural voice commands.

---

# 📌 About

Advance Voice Assistant is a Python-based virtual assistant designed to improve productivity through voice interaction. It combines speech recognition, desktop automation, web services, and Windows system utilities into a single application for hands-free computer control.

# 📦 Installation

```bash
# Clone Repository
git clone https://github.com/Anex-priyanshu/Advance-Voice-Assistant.git

# Open Project Folder
cd Advance-Voice-Assistant

# Install Required Packages
pip install -r requirements.txt

# Run Project
python Jett.py
```
